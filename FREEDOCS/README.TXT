### FreeDOS 1.3 Installation media.

* FD13FLOP.IMG - Basic FreeDOS installation boot floppy image.
* FD13LITE.IMG - Plain DOS system only installation USB stick image.
* FD13FULL.IMG - Plain DOS system and Full install USB stick image. (Also, includes may extra and uninstalled packages)
* FD13LGCY.ISO - Same as FD13FULL but using the classic El Torito Boot CD-ROM format.
* FD13LIVE.ISO - FreeDOS Live CD along with support for a Plain and Full installation.
